# Lapsell - クリエイター作業時間オークション

Lapsellは、クリエイターが自身の将来の作業時間を「作業枠」として販売し、支援者が入札して購入できるプラットフォームです。クリエイターは作業中にタイムラプス動画を撮影し、作品リリース後に落札者のみに視聴を解放する仕組みです。

## 主な機能

- 🎨 **作業枠オークション**: クリエイターが作業時間を出品、ユーザーが入札
- 🎬 **タイムラプス動画**: 作業過程の記録・視聴機能
- 👥 **複数人落札**: 最大6名まで同時落札可能
- 📱 **モバイルファースト**: スマホアプリのようなUI/UX
- 🔄 **ユーザー切り替え**: デモ用の複数アカウント機能

## 技術スタック

- **フロントエンド**: React 18 + TypeScript
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS v4
- **UIコンポーネント**: Shadcn/ui
- **アイコン**: Lucide React

## セットアップ

### 前提条件
- Node.js 18.0.0 以降
- npm または yarn

### インストール

1. リポジトリをクローン
```bash
git clone [repository-url]
cd lapsell
```

2. 依存関係をインストール
```bash
npm install
```

3. 開発サーバーを起動
```bash
npm run dev
```

4. ブラウザで http://localhost:5173 を開く

### その他のコマンド

```bash
# プロダクションビルド
npm run build

# プレビュー（ビルド後）
npm run preview

# リンティング
npm run lint
```

## デモの使い方

### 基本的な流れ

1. **ユーザー切り替え**: 設定画面でアカウントを切り替えて異なる視点を体験
2. **作業枠出品**: 右下の「+」ボタンから新規作業枠を出品
3. **入札**: 作業枠をタップして詳細画面で入札
4. **オークション終了**: クリエイターがオークションを終了
5. **作業開始**: 落札後、クリエイターが作業を開始
6. **動画アップロード**: テスト機能で動画ファイルをアップロード
7. **作業完了**: 作業を完了してタイムラプス動画を登録
8. **動画解禁**: クリエイターが動画を落札者に解禁
9. **視聴**: 落札者が動画を視聴

### 主な画面

- **ホーム**: 作業枠一覧表示
- **作業枠詳細**: 入札、動画視聴、作業進行管理
- **プロフィール**: 購入・出品履歴
- **設定**: ユーザー切り替え、デモ説明
- **新規出品**: 作業枠の出品フォーム
- **クリエイター詳細**: 特定クリエイターの作品一覧

### 複数人落札機能

- 最大6名まで同時落札可能
- 上位入札者が自動的に落札者に決定
- 全落札者が動画視聴権を獲得

## 開発について

### スタイリング
- Tailwind CSS v4を使用
- 基本的なタイポグラフィはglobals.cssで設定済み
- スクロールバーは全画面で非表示化済み

### 状態管理
- React Contextでアプリケーション状態を管理
- 作業枠データ、ユーザー情報、画面遷移を統合管理

### レスポンシブ対応
- モバイルファーストで設計
- タブレット・デスクトップでも適切に表示
